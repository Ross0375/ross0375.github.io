About Bootstrap

Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery.

Components

Bootstrap is fantastic for fast development of professional looking websites without knowledge of javascript, just a basic understanding of html and css is required to start making websites. One of my favorite components in Bootstrap is the carousel, showing multiple images in one box, slowly rotating throughout, without any knowledge of javascript required.
