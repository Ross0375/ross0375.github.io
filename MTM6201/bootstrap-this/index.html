About Bootstrap

Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery.

Set Up

Bootstrap is very easy to set up and get it running on your website, all you need to do is copy and paste this handy-dandy code when you're starting your website and you are good to go! No need to touch any other files (Unless you want to customize some minor things)

                      
<!doctype html>
  <html lang="en">
    <head>
      <!-- Required meta tags -->
      <meta charset="utf-8">
      <meta name="viewport" content="width=device-width, initial-scale=1">
                  
      <!-- Bootstrap CSS -->
      <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0-beta3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-eOJMYsd53ii+scO/bJGFsiCZc+5NDVN2yr8+0RDqr0Ql0h+rP48ckxlpbzKgwra6" crossorigin="anonymous">
                  
      <title>Hello, world!</title>
    </head>
    <body>
       <h1>Hello, world!</h1>
                  
      <!-- Optional JavaScript; choose one of the two! -->
                  
      <!-- Option 1: Bootstrap Bundle with Popper -->
      <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0-beta3/dist/js/bootstrap.bundle.min.js" integrity="sha384-JEW9xMcG8R+pH31jmWH6WWP0WintQrMb4s7ZOdauHnUtxwoG2vI5DkLtS3qm9Ekf" crossorigin="anonymous"></script>

Grids Grids Grids!

Grids are super simple with bootstrap with built in responsiveness right out of the box. All you have to do is set give the container a .row tag, and then set give the children of the row tag their column values. All it takes is a simple .col-6 and a .col-lg-12 and you're off to the races with responsiveness.

If you resize you will notice that everything will move to a different position. This is all done without even touching any css.

This is the accordion! It's a way to hide text under a small dropdown for those to read when they're interested, or to just skip over. You can choose whether it's open by default or not, along with other minor options like removing the border and changing the colour.

Components

Bootstrap is fantastic for fast development of professional looking websites without knowledge of javascript, just a basic understanding of html and css is required to start making websites. One of my favorite components in Bootstrap is the carousel, showing multiple images in one box, slowly rotating throughout, without any knowledge of javascript required.

Cards!

Cards are also a component built into Bootstrap. They're a great way to display small packets of information. YOu can put almost anything into them and they will work! Very easy to use and understand

You don't even need images.
Just text will work too.

As you can see they are very flexible and can be used for whatever you want them to be used for. Just a few lines of code and you're ready to go.

Modals!
It's a pop-up!

Modals can be used anytime anywhere, you can make so they pop-up before someone agrees to something, or just make sure the user understands what they are doing. You can center it on the page or just have it at the top like I do. You can have as much text as you like in here because it is scrollable too. Even images work! The world is your oyster with modals.
